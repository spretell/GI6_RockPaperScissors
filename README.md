A cutely designed and interactive rock, paper, scissors game featuring a pastel theme, animated cat paws, GIF popups, and a scoreboard!
## *How to Play*
1. The user must type **rock**, **paper**, or **scissors** in the text box on top, then hit **play!**
2. After hitting play, kitty paws will appear showing your choice on the left and the computer's choice on the right.
3. Alongside that, a popup will show if you win, lose, or tie with a cute cat gif.
4. Check your scores with the scoreboard down below.
5. Reset whenever you want to start fresh!
## *Features*
- Animated cat paw movements with CSS & JS
- Smooth CSS @keyframes animations
- Cute cat gif popups along with play results
- A scoreboard that tracks user's wins, losses, or ties
- A reset button to clear all previous scores
- Responsive design using mobile-first design
- Comments throughout code for easy reading/understanding
## *Coding Files Used*
- **HTML** for structure and content
- **CSS** for styling structure and adding keyframe animations
- **JavaScript** for the game process
